# Maisha Umama - Portfolio

A seamless, modern portfolio built with React, Vite, and Tailwind CSS.

## Features

- **Modern UI/UX**: Clean, responsive design with smooth transitions.
- **Project Showcase**: Highlight key projects with detailed information.
- **About Me**: A section dedicated to the creator's background and skills.
- **Contact**: Integrated contact form for easy communication.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: GSAP & Framer Motion
- **Icons**: Lucide React
- **Components**: Radix UI & Shadcn/UI

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/mai-umama/maisha-umama.portfolio.git
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
